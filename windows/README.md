# Windows deploy tools

PuTTY + WinSCP automation for uploading and building Nexlify on your VPS.

- **Start here:** [DEPLOY-WINDOWS.md](./DEPLOY-WINDOWS.md)
- **Quick run:** double-click [deploy.bat](./deploy.bat)
- **Config:** copy `deploy.config.example.json` → `deploy.config.json`

This folder stays on your PC only and is not synced to Linux.
